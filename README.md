#二维码生成+ZIP打包
##
###二维码生成
引入QRcode类，'library/QRcode.php'

生成二维码和zip打包是分开写的

网上QRcode生成二维码教程很多

根据需求设置好几个属性就可以

*$text* : 二维码的值

*$filename* : 保存路径

*$px* : 清晰度menu(L(7%), M(15%), Q(25%), H(30%))  正常二维码应该为L ，没有具体调查过

*$size* : 图片尺寸 设置为10的时候尺寸为270*270

*$quality* ： 图片质量 （一直用的默认，不是太了解）

*$set* : 自己设置的变量名，原来叫啥忘了，控制是否显示，如果只是想生成到路径里而不显示的话设置成false就好了
